Seletores

seletor de tag

Ao usar o seletor tipo tag em H1 e H2 todos os elementos deste tipo vão herdar os estilos definidos pelo codigo CSS

seletor de tipo

o seletor tipo #id deve ser usado uma única vez no arquivo! Cada elemento pode ter apenas um id. Cada pode ter apenas um elemento com aquelo ID

Seletor de classe

Qualquer informação de estilo que precise ser aplicada a múltiplos emelentos em uma página deve ser feita com uma classe. Você pode usa várias classes para um mesmo elemento
